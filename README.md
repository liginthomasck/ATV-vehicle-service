# ATV

<br/>
<br/>

 <br/>
### 1.Admin <br/>
-- manage mechanics <br/>

 <br/>
### 2.Users <br/>
-- add service <br/>
-- book vehicles <br/>
-- view service status <br/>
 <br/>
### 3.Mechanics <br/>
-- edit service status <br/>
-- set type of services <br/>
