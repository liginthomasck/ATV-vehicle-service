# ATV

A web based application developed in Python-Django for a particular vehicle manufacturing company.


## Modules

### Admin
-- manage mechanics
-- manage registered users
-- view services
-- view bookings
-- add mechanics
-- add vechicles
-- provides login credentials

### Users
-- add service
-- book vehicles
-- view service status

### Mechanics
-- edit service status
-- set type of services
